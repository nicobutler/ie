Introduction

Since I was a kid, everything about technology, computers, the internet, caught my eye. My first adventure in the tech world was when I was 16, a friend and I started a little hobby, which involved upgrading and repairing friends’ and acquaintances’ computers. All of this, we did because we enjoyed it.

Nowadays I enjoy writing code (a.k.a programming) in my free time, in particular building webpages and mobile apps. It’s a way to express my creativity and sometimes build something useful for others.

I know it’s related to my everyday job and my career. But, instead of building software for others and with deadlines, in my off times I like to write code as a hobby. Creating something from scratch, and watching it work can be quite fulfilling.

Next

How it contributes to my personal and professional development

Personally, it’s a way to express myself and helps me develop my creativeness. Writing software is fun and quite enjoyable. There is a significant amount of gratification to see your ideas come to life. Everything is possible!

Professionally, this activity can be very useful, in a sense what is done when someone builds software, is to take one’s thoughts and conceptualize them into a working design. Not only in software endeavors, but also in every other professional environment.
